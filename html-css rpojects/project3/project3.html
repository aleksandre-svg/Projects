The Circulatory System

The circulatory system is responsible for transporting blood throughout the body. It consists of the heart, blood vessels, and blood.

The Digestive System

The digestive system is responsible for breaking down food, absorbing nutrients, and eliminating waste.

The Nervous System

The nervous system controls both voluntary actions (like moving muscles) and involuntary actions (like breathing). It is made up of the brain, spinal cord, and nerves.

The Skeletal System

The skeletal system provides structure to the human body. It protects internal organs and enables movement.
